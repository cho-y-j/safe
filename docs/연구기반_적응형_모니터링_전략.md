---

## 1. 핵심 개념: 적응형 샘플링 (Adaptive Sampling)

상황에 따라 센서 측정 주기를 자동으로 변속하여 배터리를 절약하면서도
이상 징후를 놓치지 않는 전략.

> "Adaptive sampling strategies optimize power use without sacrificing performance.
>  Modern wearables adjust sensor frequencies based on energy availability and
>  the user's activity level — lowering sampling rates during rest and ramping
>  them up during physical activity or when precise data is crucial."
>
> — Flexible Wearable Heart Rate Monitoring System and Low-Power Design (PMC, 2025)

## 2. 연구 결과 요약

### 2.1 배터리 절약 효과

| 전략 | 절약 효과 | 출처 |
|------|---------|------|
| Adaptive Heartbeat Locked Loop | **3.3배** 전력 절감 | MDPI Electronics 2021 |
| 이벤트 기반 전환 | **80%** 전력 절감 | Energy-Saving Wearables 2024 |
| 동적 샘플링 (TAPIR 알고리즘) | 신호 분해 + 희소 재구성 최적화 | PMC 2025 |

### 2.2 샘플링 주파수와 정확도

| 샘플링 주파수 | 정확도 | 배터리 영향 |
|-------------|--------|-----------|
| 100~200 Hz (연속) | 최고 (임상 수준) | 매우 높음 |
| 40~50 Hz | 20% 편향 발생 | 중간 |
| 1회/60초 (간헐) | 심박수 추적 충분 | 매우 낮음 |
| 이벤트 기반 적응형 | 정확도 유지 + 저전력 | **최적** |

> "Hardware–algorithm co-design is a crucial approach to achieving
>  low-power heart rate signal acquisition."
>
> — PMC 2025, Flexible Wearable HR Monitoring

### 2.3 가속도계의 역할

- 가속도계는 **초저전력 (~0.5mW)** 으로 항상 켜둬도 배터리 영향 미미
- 움직임 변화를 감지하여 심박 센서 측정을 트리거
- 낙상 감지 (자유낙하 + 충격 패턴)도 가속도계로 수행

### 2.4 연속 vs 간헐 모니터링

| 방식 | 장점 | 단점 |
|------|------|------|
| 연속 (Continuous) | 이상 놓치지 않음 | 배터리 4~8시간 |
| 간헐 (Intermittent) | 배터리 48시간+ | 측정 사이 이상 놓칠 수 있음 |
| **적응형 (Adaptive)** | **배터리 + 정확도 모두 확보** | 알고리즘 복잡도 증가 |

## 3. SafePulse 적용: 5단계 적응형 모니터링

### 3.1 단계 정의

| 단계 | 상태 | 심박 주기 | 서버 전송 | 예상 배터리 |
|------|------|----------|---------|-----------|
| **1. 안정 감시** | 앉아있음, 움직임 없음 | 60초 | 5분 | 48시간 |
| **2. 활동 감시** | 걷기, 작업 중 | 30초 | 2분 | 36시간 |
| **3. 변화 감지** | 심박 ±15 급변 | 15초 | 30초 | 24시간 |
| **4. 경보 근접** | 경보 범위 80% 도달 | 5초 | 10초 | 12시간 |
| **5. 경보 초과** | 경보 범위 초과 | 1초 | 즉시 | 4시간 |

### 3.2 단계 전환 트리거

```
[가속도 변화] → 안정 ↔ 활동 전환
  가속도 deviation > 1.5 = 활동
  가속도 deviation < 0.5 (10초 지속) = 안정

[심박 급변] → 변화 감지 모드
  이전 측정 대비 ±15 bpm 이상 변화 = 주기 단축

[경보 근접] → 고속 모드
  (현재 심박 - 안정 평균) > (경보 범위 × 0.8) = 5초 모니터링

[경보 초과] → 최고속 + 알림
  경보 범위 초과 = 1초 모니터링 + 본인 알림
  → 5초 내 확인 없음 → P2P BLE 경보 + 서버 전송

[정상 복귀] → 저전력 복귀
  심박이 경보 범위 70% 이내로 복귀 = 단계 하향
```

### 3.3 특허와의 관계

이 설계는 특허 "BLE-UWB 하이브리드 기반 적응형 웨어러블 안전장치"의
**제1 모드(저전력) ↔ 제2 모드(고출력) 이중 모드 전력 관리**와 일치함.

- 제1 모드 = 단계 1~2 (저전력, BLE 저출력)
- 제2 모드 = 단계 4~5 (고출력, BLE 고출력 + UWB 기동)
- 전환 조건 = 단계 3 (변화 감지)
